Dev C++ 是一款免费开源的 C/C++ IDE，内嵌 GCC 编译器（GCC 编译器的 Windows 移植版），是 NOI、NOIP 等比赛的指定工具。Dev C++ 的优点是体积小（只有几十兆）、安装卸载方便、学习成本低，缺点是调试功能弱。

安装 Dev C++ 跟安装普通软件一样，远没有安装 VS 那么复杂。

截止到2017年06月07日，Dev C++ 的最新版本是 5.11。
